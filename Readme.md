# reflection
Working through the programming logic problems helped me understand the basics of how programs are structured. In Level 1, I practiced simple input, process, and output tasks. These problems showed me that even basic calculations like averages or totals can be automated with clear step-by-step logic.

In Level 2, I learned how to handle multiple inputs and combine them with more than one operation. Problems like computing earnings or grades taught me to break down real-world scenarios into smaller formulas and then connect them logically. This made me realize how programming can solve everyday tasks more efficiently.

Finally, Level 3 introduced decision-making with IF/ELSE logic. Here, the program doesn’t just calculate—it chooses outputs based on conditions. Whether checking violations, classifying stock changes, or identifying even and odd numbers, I saw how decisions make programs dynamic and interactive. Overall, these exercises built a strong foundation for more advanced programming concepts.
